AP Computer Science Vocab


Section 1 Vocabulary

  1. Stage: the stage is what the program is performed on
  2. Costume: the costume is the design of the sprite and/or background
  3. Sounds: the sound is the effect that you can put in a script to create a sound effect to match up with the program
  4. Pair Program: a program that is created by two people
  5. Camera: something that takes pictures
  6. Allow: to give permission to
  7. Save: to make sure that the program doesn’t get deleted by the computer after you finish your work
  8. Import: to transfer one file into another program to make it available for use
  9. Select: a tool that allows people to grab something in their edits and move it
  10. Delete: done through the select tool, delete lets you get rid of any program, costume, or sprite that is unwanted
  11. Info: the information shown when a script, sprite, background, or costume is right clicked
  12. Event: the trigger that starts the script that you want
  13. If Then: a block in Scratch that allows that expression to run if the program is true

Section 2 Vocabulary

  1. Packets: A chunk of data transmitted as a unit over a network, sometimes in smaller bursts called frames.
  2. Router: A piece of hardware responsible for dictating the flow of packets over a network.
  3. ISP (Internet Service Providers): Internet Service Providers make Internet access available to users.
  4. Protocol: An agreed-upon method for conducting a communication.
  5. IPv4: The Internet Protocol version 4 is the dominant protocol for routing traffic on the Internet, specifying “to” and “from” addresses using a dotted decimal such as “122.45.255.0”.
  6. Dotted Decimal Notation: A representation of numerical data in which groups of base-10 numbers are separated by dots.
  7. IPv6: The Internet Protocol version 6 provides a large number of new addresses to route Internet traffic, using “from” and “to” addresses written as colon-hexadecimal notation, such as “fe80::42:acff:feaa:1bf0”.
  8. Domain Name System(DNS): A hierarchical system for naming resources on the Internet.
  9. Name Servers: A computer dedicated to providing responses to requests for Domain Name information.
